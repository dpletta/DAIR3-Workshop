# FOO prototypes

Earlier exploratory scripts kept for reference. These are NOT part of the
active FOO workshop suite; the four maintained entry points live one folder
up:

- `agentClaude.py` — single Anthropic agent (now wraps `single_agent_gui.py`)
- `agentGPTGUI.py` — single OpenAI agent (now wraps `single_agent_gui.py`)
- `agentGoogleGUI.py` — single Gemini agent (now wraps `single_agent_gui.py`)
- `foo_gui.py` — multi-agent tabbed FOO workflow

Anything in this directory predates the unified Provider/Model selector,
the RAG knowledge base, and the unified file router. Treat these files as
historical artifacts: feel free to read them, but do not depend on them.
